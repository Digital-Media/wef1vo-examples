CSS Grid Showcase

1. display: grid (Block-Ebene)

Der folgende Container nimmt standardmäßig die volle verfügbare Breite ein:

A
B

Dadurch wird der Textfluss vor und nach dem Grid unterbrochen (Zeilenumbruch).

2. display: inline-grid (Inline-Ebene)

Hier ist ein Fließtext, und mittendrin befindet sich X Y ein Inline-Grid. Es fließt mit dem Text und erzwingt keinen Umbruch. Dank vertical-align: middle sitzt es mittig auf der Zeile.

3. Subgrid (Verschachtelung)

Das (äußere) Main-Grid hat 5 Spalten. Das mittlere Element (grau gestrichelt) ist ein Kind, das sich über 3 Spalten spannt. Dessen Kinder (die roten Boxen) nutzen subgrid, um sich exakt an den Spalten des Main-Grids auszurichten.
